MelonBot

MelonBot is a discord bot, designed simply for fun, by accessing a crazy amount of APIs, written in python. It basically just retrieves troves on info from the web onto your server.

I think it has ~28 functions (not all are API related of course)

A list of the functions is detailed below (this list may not be comprehensive):

  • Cool Wikis: gets a link to a random wiki from the Wikipedia:Unusual_articles page
  • Wiki: searches wikipedia for an input value
  • ISS: Gets the current latitude and longitude of the International Space Station
  • Random Number: Gets an interesting fact related to a random number
  • E: Calculates the number e from a given input using the equation (1 + 1/n)^n=e
  • ID: Gets your discord user ID
  • Author: Returns facts about an inputted author
  • Insult: Has the bot insult you
  • Affirmation: Has the bot send you an affirmation
  • Advice: Gives you some advice for life
  • Nasa: Gets a beautiful image from NASA
  • Quote: Gets a random (real) quote
  • Food: Gets a picture of some food
  • Number: Gets info about a number
  • Ping: Gets the current ping of the bot
  • Free games: Gets some info about a free game
  • Magic: Gets data about a random Magic: The Gathering card
  • Bedwars: Gets minecraft bedwars stats about an inputted user
  • Skywars: Gets minecraft skywars stats about an inputted user
  • Shibu: Gets a random shibu inu picture
  • Fox: Gets a random fox picture
  • Cat: Gets a random cat picture
  • Tronalddump: Gets a donald trump quote
  • Programming joke: Gets a random programming joke
  • General Joke: Gets a random joke
  • Dad joke: Gets a random dad joke

The only real way to get MelonBot running on your server is to download the repo, and then set it all up yourself. You will need to generate your own API keys (I think just for NASA and Hypixel), alongside make your own discord application, and get its token. This is because I seldom actually host MelonBot, due to my computer not being good, and I don't wanna pay for cloud stuff (and putting it on repl feels like cheating).
