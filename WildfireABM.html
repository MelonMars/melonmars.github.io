Wildfire ABM

Wildfire ABM is an agent based model based on the paper "A cellular automata model for forest fire spread prediction: The case of the wildfire that swept through Spetses Island in 1990", which is about an agent based model for simulating a wildfire in Greece.

It uses the python MESA library for visualization, diagnostics, parameter modifying, and for providing a WebUI.

It incorporates many features:

  • Spotting (when a fire jumps through the air)
  • Slopes
  • Wind
  • Random terrain generation
  • Various species and densities
  • Non-burning tiles (e.g. cities, roads)
